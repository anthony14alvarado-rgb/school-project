AP Biology: Unit 4 Statistics

Chi-Square Analysis ($\chi^2$) is used to determine if there is a significant difference between the observed and expected frequencies in one or more categories. In the case of Mendelian genetics, we often use the formula:

$\chi^2 = \sum \frac{(o-e)^2}{e}$

Where 'o' is the observed result and 'e' is the expected result. This analysis helps biologists reject or fail to reject the null hypothesis. (Keep this screen up if a supervisor approaches).
